# Skills index

The skills in this OS, what each does, and what triggers it. When you add a skill, add a row here.

| Skill | What it does | Say |
|---|---|---|
| setup-os | First-run onboarding: interviews you, populates me.md and context, pulls a first map of your tools | "set up my OS", "onboard" |
| save-to-os | Saves a note, fact, or decision into the OS folder as a new dated file | "save this to my OS", "log this decision" |
| question-mode | Helps you think through or build something by asking one question at a time | "use question mode" |
| skill-creator | Helps you build a new skill and record it here | "create a skill" |
| os-audit | Weekly Four Cs health check, flags what is thin or leaking | "run my audit" |
| os-level-up | Weekly Three Ms pass, turns one manual task into a skill | "level up", "what should I automate" |

## How to add a skill

Build it with the **skill-creator** skill, upload it at Customize, Skills, Create skill, then add its row above. This file you edit by hand in Google Docs (it is your Drive), or ask Claude to draft the new row and you paste it in. The Drive connector can create files but not edit them, so the index is kept current by you, not automatically.
