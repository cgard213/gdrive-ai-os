# How the inbox works

This folder is for fast capture. A thought, a lead, a thing you decided in the car, a task you do not want to lose. One line is fine. Do not file it, do not format it, just get it down before it is gone.

## The pattern

- Drop a note here any time, from any path. It does not have to be sorted.
- Name it loosely, the date plus a few words is plenty, for example `2026-06-24-callback-the-listing-lead.md`. Or just paste into one running note if that is faster for you.
- Once a week, the audit reads this folder with you and sorts each item: into the right path's `overview.md`, saved as a decision, turned into a skill, or dropped.

The inbox should end most weeks close to empty. It is a holding pen, not a home.
