# Skills index

The skills in this OS, what each does, and what triggers it. When you add a skill, add a row here.

| Skill | What it does | Say |
|---|---|---|
| onboard | First-run setup: asks your platform and paths, interviews you, populates me.md and each path's overview, maps your tools | "onboard me", "set up my OS" |
| save-to-os | Saves a note, fact, or decision into the right path's decisions folder as a new dated file | "save this to my OS", "log this decision" |
| question-mode | Helps you think through or build something by asking one question at a time | "use question mode" |
| skill-creator | Helps you build a new skill and record it here | "create a skill" |
| os-audit | Weekly Four Cs health check, triages the inbox, flags what is stale | "run my audit" |
| os-level-up | Weekly Three Ms pass, turns one manual task into a skill | "level up", "what should I automate" |

## How to add a skill

Build it with the **skill-creator** skill, then add it on your platform: upload a Skill (Claude), fold it into your Gem (Gemini), or your Custom GPT (ChatGPT). Then add its row above.

You edit this file by hand (it is your Drive), or ask the AI to draft the new row and you paste it in. On Claude the connector can create files but not edit them, so the index is kept current by you, not automatically.
