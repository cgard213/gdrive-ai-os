# Connections

The tools Claude can reach, and what each one is for. Fill this in as you connect them, or let "set up my OS" map it for you.

| Tool | What it holds | Claude uses it for |
|---|---|---|
| Google Drive | Files and documents | Finding and reading anything on file |
| Gmail | Email | Catching up on a thread, drafting replies |
| Google Calendar | Schedule and appointments | The day, the week, what is booked |
| (Your CRM) | Contacts, pipeline, deals | The source of truth for any contact or deal |
| (Your boards) | Projects and tasks | Open work, who owns what, what is due |
| (Your design tool) | Marketing and designs | Finding and starting designs |

## Notes

- A tool with no built-in connector (many CRMs) can be bridged through Zapier. Set up a server at mcp.zapier.com, switch on the actions you want, and add the URL as a custom connector. Zapier exposes only the actions you turn on, not the whole tool.
- Anything you mention often but is not connected is a blind spot. The weekly audit flags these.
