# What to paste into the project, gem, or custom GPT

Create a workspace in your AI and paste the block below into its instructions field. Keep it short. The depth lives in the Google Drive folder, so this only has to point there.

- **Claude:** Projects, New project, then paste into custom instructions. Add the OS folder to the project.
- **ChatGPT:** Projects, new project (or a Custom GPT), then paste into instructions. Link the OS folder.
- **Gemini:** New Gem, then paste into instructions. Reference the rollup files (`me.md`, `_shared/`, each `overview.md`), not the whole folder.

---

My operating system lives in a Google Drive folder. On any task, read `instructions.md` in that folder first, then `me.md`. They are your standing context: who I am, how I work, what tools you can reach, the paths I run, and the decisions I have made.

When a task is about a specific area, read that path's `overview.md` and check its `decisions/` before assuming. Use the skills in `_shared/skills/INDEX.md` when they fit. When I ask you to save, log, or record something, write it as a new dated note in the right path's `decisions/`, and never overwrite an existing note.

Default to doing the task, not explaining it. If one fact is missing, ask one specific question and stop.
